WhatsApp Bot

Your AI-powered store assistant — message it to get instant store insights
What the bot does

WhatsApp your bot number and it replies instantly with live data from your myvendor store. Works 24/7, no dashboard needed.

orders→ Today's count and pending order list
revenue→ This month's earnings
products→ Inventory count and out-of-stock alert
stats→ Full store overview in one message
Any sentence→ AI answers in plain English (requires Anthropic key)

One-time setup (~10 min)
1
Create a Meta Developer App
Go to developers.facebook.com/apps → New App → Business → add the WhatsApp product.
2
Get your credentials
From WhatsApp → API Setup: copy the Phone Number ID and generate a Permanent Access Token (or use a System User token for production).
3
Add environment variables to Vercel
Vercel dashboard → your project → Settings → Environment Variables:
WA_VERIFY_TOKEN = any secret string you choose (e.g. "mv-bot-2026")
WA_ACCESS_TOKEN = permanent token from Meta
WA_PHONE_NUMBER_ID = from Meta Developer Console
ANTHROPIC_API_KEY = from console.anthropic.com (optional — enables AI replies)
4
Configure the webhook in Meta
WhatsApp → Configuration → Webhooks:
• Callback URL: https://myvendor.qzz.io/api/whatsapp
• Verify token: the value you set in WA_VERIFY_TOKEN
• Subscribe to: messages
5
Match your WhatsApp number in Settings
The bot identifies you by your phone number. Go to Settings and confirm the WhatsApp number you'll message from is saved there (include country code, no +).
6
Test it!
Message your WhatsApp Business number with "hi" — you should instantly receive a reply with the command menu. If not, check that Vercel has redeployed after adding env vars.
Note: The bot only responds to WhatsApp numbers that match a vendor account. Other numbers receive a sign-up prompt. Make sure SUPABASE_URL and SUPABASE_SERVICE_ROLE_KEY are already set in your Vercel environment (they should be from your initial setup).
